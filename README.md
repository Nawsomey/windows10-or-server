How to run the powershell script:
1. Open PowerShell as an Administrator.
2. Type Set-ExecutionPolicy Unrestricted.
3. Press Enter.
4. Type A.
5. Run the PowerShell script.
6. Once finished, type Set-ExecutionPolicy Restricted.
7. Press Enter.
8. Type Exit.
